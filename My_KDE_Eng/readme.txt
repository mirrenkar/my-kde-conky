============================================================
 EN: Installation & Usage (My_KDE_Eng)
============================================================
1. Copy the folder "My_KDE_Eng" to: $HOME/.config/conky
2. Open a terminal inside the "My_KDE_Eng" folder.
3. Run the script:

   ./start.sh              - Starts Conky widgets
   ./start.sh -change      - Starts Conky widgets and sets the wallpaper (from wallpaper.png)
   ./start.sh stop         - Stops all running Conky processes
   ./start.sh restart      - Restarts the theme
   ./start.sh restart -change - Restarts the theme and sets the wallpaper (from wallpaper.png)
   ./start.sh boot         - Delayed start (best for KDE Autostart settings)
   ./start.sh boot -change - Delayed start with wallpaper change

============================================================
 RU: Установка и Использование (My_KDE_Rus)
============================================================
1. Скопируйте папку "My_KDE_Rus" в: $HOME/.config/conky
2. Откройте терминал внутри папки "My_KDE_Rus".
3. Запустите скрипт:

   ./start.sh              — Запуск виджетов Conky
   ./start.sh -change      — Запуск виджетов Conky и установка обоев (из wallpaper.png)
   ./start.sh stop         — Полная остановка всех процессов Conky
   ./start.sh restart      — Перезапуск темы
   ./start.sh restart -change — Перезапуск темы с установкой обоев (из wallpaper.png)
   ./start.sh boot         — Запуск с задержкой (для автозагрузки в KDE)
   ./start.sh boot -change — Запуск с задержкой и установкой обоев
