# My KDE Conky Theme

Стильный и информативный набор виджетов **Conky** для рабочего стола **KDE Plasma**. Этот пак включает в себя системный мониторинг, погоду, сетевую активность и музыкальный плеер, адаптированные под темную тему с яркими оранжевыми акцентами.


## Особенности
* **Gotham**: Большие стильные часы и базовая информация о системе (CPU, RAM, HDD).
* **Process**: Детальный мониторинг загрузки процессора и оперативной памяти с выводом топ-процессов.
* **Network**: Мониторинг скорости загрузки/отдачи (WLAN), SSID и внешнего IP.
* **Weather**: Прогноз погоды для вашего города (по умолчанию Бруклин, NY) через сервис wttr.in.
* **Music Player**: Управление и отображение текущего трека с помощью `playerctl`.
* **Автоматизация**: Скрипт для автоматической установки обоев и запуска всех виджетов одной командой.
